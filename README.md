# AI-Powered Personal Finance Tracker

A full-stack web app where users log expenses and an AI assistant analyzes spending patterns and answers natural language questions.

## 🌐 Live Demo
[https://ai-finance-tracker-orpin.vercel.app](https://ai-finance-tracker-orpin.vercel.app)

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Recharts
- **Backend:** Node.js, Express, PostgreSQL (Supabase)
- **AI:** Groq API (Llama 3)
- **Auth:** JWT + bcrypt
- **Deployment:** Vercel (frontend) + Railway (backend)

## ✨ Features
- JWT-based user authentication
- Add, view, and delete transactions by category
- Budget tracking with progress bars
- Pie and bar charts for spending analytics
- AI chat assistant — ask questions like "How much did I spend on food?"
